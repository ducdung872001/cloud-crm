## Quy ước

- **Mã yêu cầu:** `UR-<MODULE>-<##>` (ví dụ: UR-CUS-01)
- **Ưu tiên MoSCoW:** **M** = Must have, **S** = Should have, **C** = Could have, **W** = Won't have (this release)
- **Trạng thái:** Draft → Reviewed → Approved → Implemented → Tested
