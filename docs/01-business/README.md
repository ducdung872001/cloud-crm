# 01-Business — Kinh doanh & Đề xuất

Folder này tập hợp toàn bộ tài liệu **mặt kinh doanh** của dự án Reborn Loyalty: bài toán khách hàng, đề xuất giải pháp, báo giá, khảo sát.

| File / Folder | Đối tượng đọc | Mô tả |
|---|---|---|
| [`executive-summary.md`](executive-summary.md) | BOD, sponsor | Tóm tắt 2 trang: bài toán, giải pháp, ROI, lộ trình |
| [`gap-analysis.md`](gap-analysis.md) | BA, PO, Architect | Đối chiếu hiện trạng Goldmem/Access/Excel/Supporter với đích, P0/P1/P2 |
| [`proposal/`](proposal/) | Khách hàng, BOD | Đề xuất giải pháp 10 phần + slide PPTX (xem `_assets/proposal/`) |
| [`pricing/`](pricing/) | **INTERNAL** — Sales, Finance | Báo giá, phân tích CAPEX, chi phí–margin |
| [`survey/`](survey/) | BA | Phiếu khảo sát Loyalty + Q&A buổi khảo sát khách hàng |
| [`customer-requirements/`](customer-requirements/) | BA, Architect | Ảnh chat yêu cầu KH + bảng Q&A đã tổng hợp |

> ⚠️ **Tài liệu pricing/ chứa thông tin INTERNAL (margin, CAPEX allocation)** — không chia sẻ ra ngoài khách hàng. File `.docx` chính đã chuyển vào `_assets/pricing/`, file markdown ở đây là nguồn để regenerate.

## Đọc theo thứ tự gợi ý

1. **Executive Summary** (3 phút) — nắm tổng thể
2. **Gap Analysis** (10 phút) — hiểu khoảng cách
3. **Proposal** (30 phút) — giải pháp chi tiết
4. **Pricing** (15 phút, internal) — phương án chi phí
