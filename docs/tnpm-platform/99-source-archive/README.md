## Cách dùng

- Khi phân tích yêu cầu cụ thể (ví dụ "phí gửi xe oto tính sao") → mở folder `sample-data/Phase1/VP&TTTM/Phí xe oto/`.
- Khi nghi ngờ spec hiện tại có sai → check lại HLD gốc `TNPM_HLD_v2.pdf`.
- Khi cần preview/share lên slide → dùng `TNPM_HLD_v2.pptx` để export riêng.

- Hợp đồng pháp lý nội bộ Reborn × TNTech (sẽ lưu riêng nếu có).
